# 🎓 Student Mark Prediction Using Machine Learning

## Project Overview

This project predicts student academic performance using Machine Learning. The prediction is based on:

* Attendance
* Study Hours
* Previous Marks
* Assignment Score

The model uses Linear Regression to estimate the final marks of a student.
